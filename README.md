🚀 ALPHA X BOT

A powerful WhatsApp bot built with Baileys, designed with an advanced EMS (Expressive Message System) architecture.

---

✨ Features

- ⚡ Fast and lightweight
- 🔄 Hot Reload (no restart required)
- 🧠 Smart command handler
- 📦 Modular plugin system
- 🛡️ Security & anti-spam systems
- 🤖 AI integrations
- 🎮 Games & entertainment
- 📥 Media downloader (YouTube, TikTok, etc.)
- 👥 Group management tools

---

📁 Project Structure

alpha/
│
├── commands/     # Bot commands
├── lib/          # Core systems (automation, security, etc.)
├── src/          # Main bot system & plugins
├── config.js     # Configuration file
├── index.js      # Entry point
├── main.js       # Core handler
├── package.json

---

⚙️ Installation

git clone https://github.com/kornosonline-dev/Alpha-X.git
cd Alpha-X
npm install

---

▶️ Run the Bot

npm start

---

🧠 How It Works

- Loads commands dynamically from "/commands"
- Uses middleware to process messages
- Supports multiple prefixes: ". ! # /"
- Handles:
  - Messages
  - Group events
  - System updates
- Uses local database for:
  - Users
  - Groups
  - Bot settings

---

📦 Technologies Used

- JavaScript (Node.js)
- Baileys (WhatsApp Web API)
- LowDB (Local Database)
- Axios / Fetch
- FFmpeg / Sharp
